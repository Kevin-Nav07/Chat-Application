# Chat-Application

## What is The Project?:

Chat application that will allow multiple users to login and chat in real time via the internet. This will allow for real-time communication. It will allow for users to sign up and log in with credentials, then add people as friends where they may chat individually or as a group.

## What is The MVP?: 

UI that allows for interaction between two users in real-time

## What is Nice to Have?

Making Friend Requests, allowing multiple user chatting, request friends, chatting with friends, login and sign up with accounts
